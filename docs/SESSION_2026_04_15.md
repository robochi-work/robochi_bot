# Session 2026-04-15: Clean Slate Theme Migration

## Summary
Full CSS theme migration from Neumorphic Steel to Clean Slate (indigo accent, flat modern cards).
Dashboard layout redesign for Employer, Worker, and Admin panels.

## Changes Made

### 1. CSS Theme Migration (`telegram/static/css/styles.css`)
- **Light theme**: background `#f1f5f9`, cards `#ffffff`, primary `#6366f1` (indigo), borders `#cbd5e1`
- **Dark theme**: background `#0f172a`, cards `#1e293b`, primary `#818cf8` (lavender), borders `#334155`
- Removed: neumorphic shadows, gradients, text-shadows, transparent backgrounds
- Added: flat shadows, border-based card separation, indigo focus rings
- Legacy CSS variable aliases preserved (`--btn-bg`, `--btn-shadow`, `--accent-dark` etc.) for inline `<style>` compatibility

### 2. Dashboard Layout System
- New shared CSS classes: `.employer-layout`, `.employer-top`, `.employer-middle`, `.employer-bottom`, `.worker-btn--compact`
- Three-zone layout: fixed top → centered middle → fixed bottom
- Applied to both Employer and Worker dashboards

### 3. Employer Dashboard (`work/templates/work/employer_dashboard.html`)
- Top (fixed): ⭐ Мій рейтинг / Мої відгуки
- Center: 🏙️ Загальна стрічка → 📝 Створити вакансію → 📋 Поточні вакансії
- Bottom (fixed): ❓ Що робити якщо? + 🛟 Допомога адміністратора
- Removed `worker-btn--accent` purple left border

### 4. Worker Dashboard (`work/templates/work/worker_dashboard.html`)
- Top (fixed): ⭐ Мій рейтинг / Мої відгуки
- Center: 📋 Мої вакансії + 🔨 Моя робота
- Bottom (fixed): ❓ Що робити якщо? + 🛟 Допомога адміністратора

### 5. Admin Dashboard (`work/templates/work/admin_dashboard.html`)
- Top: Адмін-панель Django button (indigo, full-width)
- Tabs: reui-style with sliding indicator, full-width, CSS animation
- Bottom: ПОШУК button pinned with `margin-top: auto`
- Tab switching via CSS classes (no inline `display` toggling)

### 6. Modal Unification
- `vacancy_form.html`: 4 validation modals migrated from inline styles to `.modal-overlay` + `.modal-card` + `.btn-primary`
- `admin_moderate_vacancy.html`: delete modal migrated to CSS classes
- `admin_search_results.html`: block modal migrated to Clean Slate vars
- `vacancy_user_list.html`: user modal migrated to Clean Slate vars
- `vacancy_my_list.html`: block modal + card styles migrated
- `admin_vacancy_card.html`: block modal + card styles migrated
- `vacancy_detail.html`: detail card, members, status badges migrated
- Shared modal classes added to `styles.css`: `.modal-sheet`, `.modal-field`, `.modal-label`, `.modal-select`, `.modal-textarea`, `.modal-actions`, `.modal-btn-submit`, `.modal-btn-cancel`, `.btn-danger`, `.modal-close`

### 7. Bug Fixes
- `admin_search_results.html`: added `{% block header %}{% endblock %}` (was showing header with Menu)
- Removed `worker-btn--accent` border-left
- Improved light theme contrast: `--background: #f1f5f9`, `--border: #cbd5e1`

## Design Principles
- Clean Slate theme from https://21st.dev/community/themes/clean-slate
- CSS variables for all colors — never hardcode
- Legacy variable aliases ensure inline `<style>` blocks work without HTML changes
- Tabs follow reui pattern: muted background container, white sliding indicator
- Dashboard layout: 3-zone flexbox (top/middle/bottom) with `min-height: 100vh`
